# WeatherApp

This app is built with React Native.

### install dependencies

Setup as described in the [React Native documentation](https://reactnative.dev/docs/environment-setup)

iOS setup:

```shell
npm i
npx pod-install
```

### start the app locally

Start the dev server

```shell
npm run start
```
